## Overview
This project integrates the Chapa Payment Gateway into a Django-based travel booking application. It supports secure payment initiation, verification, and payment status tracking.

## Features Implemented
- Payment model for tracking transactions
- Payment initiation using Chapa API
- Payment verification endpoint
- Payment status handling (Pending, Completed, Failed)
- Secure API key handling via environment variables
